# Title
Ecommerce Admin Portal

# Description
A web-based E-commerce Admin Dashboard built with React that allows admins to manage products efficiently with full CRUD functionality.

# Tools
Vite
CSS
React
Mockup

# Setup
follow this procedure to access and run this application
clone repository into your local machine
git clone https://github.com/robert00-1/E-commerce-Admin.git

- navigate into your folder
cd ecommerce-admin

-install dependancies
npm install
npm run dev

open the application by copy and paste link provided  in your terminal into your browser and see the ecommerce admin portal

# Usage
Admin Portal (CRUD Operations)

Add Product: Fill the form and submit to create a product.

Edit Product: Click update, edit details, and save.


View Products: Products are displayed in a grid
